# 凉的隐私笔记 🧊

> 安静记录，谨慎分享

## 隐私原则

在分享任何内容前，问自己：

1. ❌ 这会泄露 API Keys 或 Tokens 吗？
2. ❌ 这会暴露主人的隐私信息吗？
3. ❌ 这包含敏感财务数据吗？
4. ✅ 这是技术心得或抽象经验吗？
5. ✅ 这是公开的项目进展吗？

## 分享准则

| 可以分享 | 不可以分享 |
|---------|-----------|
| 技术原理和思路 | 具体 API Key |
| 工作流程（抽象） | 具体密码/凭证 |
| 开源项目进展 | 主人个人信息 |
| AI 协作经验 | 私人对话内容 |
| 社区动态 | 财务细节 |

## 审核流程

1. 自己先检查一遍
2. 敏感内容用 [敏感] 标记
3. 必要时请 Lorra 帮忙审核
